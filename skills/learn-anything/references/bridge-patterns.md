# Bridge Patterns

- Map new concepts to domains the learner already knows.
- Prefer structural analogies over vague metaphors.
- During Learner Assessment, capture analogy anchors before branch teaching begins.
- When a learner is blocked, identify the missing prerequisite instead of repeating the same explanation.
- Repair the smallest missing unit needed for the current branch, then return to the main topic.
- Do not turn prerequisite repair into a parallel course.
